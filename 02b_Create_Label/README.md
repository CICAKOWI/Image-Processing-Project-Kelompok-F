# 02b_Create_Label — Membuat Dataset Label (.npy)

## Tujuan
Membuat label one-hot encoding yang sesuai dengan dataset input.

## Alur
1. Jalankan `create_dataset_(labels).py`
2. Label dibuat otomatis berdasarkan jumlah kelas dan sampel

## Script
| File | Fungsi |
|------|--------|
| `create_dataset_(labels).py` | Membuat one-hot labels (perlu disesuaikan jumlah kelas) |

## Output
- `labels_*.npy` — label siap dipasangkan dengan input dataset
