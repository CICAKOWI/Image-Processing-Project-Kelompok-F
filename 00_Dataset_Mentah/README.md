# 00_Dataset_Mentah — Dataset Mentah

## Tujuan
Folder ini berisi foto mentah Aksara Sunda yang sudah diatur kontrasnya secara manual.

## Struktur
| Folder | Isi |
|--------|-----|
| `hasil/` | Gambar siap diproses ke tahap preprocessing |
| `jangan_dulu/` | Backup gambar yang tidak terpakai |

## Alur
1. Foto mentah ditempatkan di folder ini
2. Kontras diatur manual
3. Gambar siap pakai ada di `hasil/`

## Output
- `hasil/*.jpg` — gambar langsung diproses ke `01_ImageProcessing/`
